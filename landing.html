A coding agent for the terminal — chapter 01

A quiet agent that writes software code.

Recode is a local-first coding agent that lives inside your terminal. Iterative tool-loop runtime, streaming responses, sandboxed shell, and a TUI that doesn't pretend to be a browser tab. Bring your own key — OpenAI, Anthropic, or anything OpenAI-compatible.

01 · Capabilities

Built like a workshop, not a chatbot.

A small, opinionated runtime: tool-loop core, streaming transport, persistent history, and a TUI that respects your terminal. No browsers. No accounts. No telemetry by default.

≋
STREAM-FIRST

Tokens land while the model thinks. No spinner staring.

The runtime yields character-by-character, surfaces in-flight tool calls, and re-enters the loop without flushing context. You watch the agent reason in real time.

03 · Three speeds of trust

You decide how loud the agent gets.

Three approval modes, persistent allowlists, and a clear visual contract for every action that touches your machine.

01
/approval-mode · approval

Cautious.

  • ✓ Reads run instantly
  • ! Edits require a yes
  • ! Bash requires a yes
  • ○ Best for unfamiliar repos
02
/approval-mode · auto-edits

Flow state.

  • ✓ Reads run instantly
  • ✓ Edits run instantly
  • ! Bash still asks
  • ○ Best for daily driver work
03
/approval-mode · yolo

YOLO.

  • ✓ Reads run instantly
  • ✓ Edits run instantly
  • ✓ Bash runs instantly
  • ○ Best for sandboxes & throwaway VMs

Install. Then forget the GUI.

Single binary, no daemon, no telemetry. Bring an OpenAI or Anthropic key and you're shipping in under a minute.

$  curl -fsSL recode.sh/install | bash
$  recode setup    // pick provider, paste key
$  recode    // or: recode "summarize the README"
